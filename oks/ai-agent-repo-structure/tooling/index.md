# Tooling

An agent is only as capable as the tools it can reach. These concepts cover
wiring external tools in through MCP, defining custom slash commands, and hooking
into an agent's workflow with scripts — all as configuration a team can commit
and share.

## Concepts

- [MCP and tool config](/oks/ai-agent-repo-structure/tooling/mcp-and-tool-config.md) — configuring MCP servers via `.mcp.json`, its scopes, and trust prompts.
- [slash commands and hooks](/oks/ai-agent-repo-structure/tooling/slash-commands-and-hooks.md) — custom commands in `.claude/commands/` and event hooks in settings.
