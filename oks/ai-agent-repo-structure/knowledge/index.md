# Knowledge

Beyond startup instructions, a repository can carry curated knowledge that an
agent reads on demand. These concepts cover shipping Open Knowledge Format
bundles alongside code, writing documentation for an agent audience, and using
frontmatter as a machine-readable contract.

## Concepts

- [OKF bundles in repos](/oks/ai-agent-repo-structure/knowledge/okf-bundles-in-repos.md) — versioned Markdown knowledge bundles that live in git next to the code.
- [docs for agents](/oks/ai-agent-repo-structure/knowledge/docs-for-agents.md) — writing documentation that both agents and humans read from the same file.
- [machine-readable metadata](/oks/ai-agent-repo-structure/knowledge/machine-readable-metadata.md) — YAML frontmatter as a structured discovery index agents parse before the body.
