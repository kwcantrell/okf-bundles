# Practices

Structure gets an agent the right context; practices make its behavior
trustworthy. These concepts cover the difference between advisory instructions
and deterministic enforcement, and how to contain what an agent can do.

## Concepts

- [determinism and reproducibility](/oks/ai-agent-repo-structure/practices/determinism-and-reproducibility.md) — hooks and pass/fail checks for behavior that must happen every time.
- [guardrails and permissions](/oks/ai-agent-repo-structure/practices/guardrails-and-permissions.md) — permission rules, modes, and sandboxing to contain agent actions.
